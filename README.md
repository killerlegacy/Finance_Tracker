# Personal Finance Tracker

A full-featured personal finance tracker built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- 📊 **Dashboard** — Monthly summary, budget progress, recent transactions, upcoming bills
- 💳 **Transactions** — Add income/expense with categories, filtering by type & month
- 🏦 **Accounts** — Track balances across Cash, Bank, Credit Card, Digital Wallet
- 🎯 **Budgets** — Set monthly category budgets with visual progress bars
- 🔁 **Subscriptions** — Manage recurring bills with due-date alerts
- 👤 **Profile** — Personal info + multi-currency support (17 currencies)
- 💾 **Persistent** — All data saved to localStorage, no backend required

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

1. Push this repo to GitHub
2. Import it in Vercel
3. Click Deploy — no environment variables needed

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Lucide React** icons
